# Quiz-Game-App
An interactive quiz application with multiple categories, timed questions, scoring system, and  performance analytics using in-memory questions and results.


You have 6 categories to choose from and 10 questions for each category. For each questions, you have 4 choices and 15 seconds to answer every number. There will be a multiplier for every correct answer streak you have. You can also have the option to skip a question but you cannot return to answer it again.
